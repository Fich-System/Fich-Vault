# Fich Vault

Documentação técnica e tracking de issues do ecossistema Fich System.

## Estrutura

- **Modulos/** — Documentação detalhada de cada módulo (backend EAV, frontend EAV)
- **MEMORY.md** — Histórico de decisões e eventos
- **Issues-Tracking.md** — Rastreamento de issues do backend
- **Issues-Front-Tracking.md** — Rastreamento de issues do frontend
- **Entidades-EAV.md** — Catálogo de entidades EAV
- **CrossReference*.md** — Mapa backend ↔ frontend
- **Pendentes.md** — Tarefas pendentes
